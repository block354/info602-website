# Server
This is the server for the web application.

## Setup

### Requirements
- [Node.js](https://nodejs.org/en) version 10 or higher.

### Installation
1. Create  a .env file in the package directory, follow the .env.example file.

### Install Dependencies
```sh
npm install
```

### Run for Development
```sh
npm run dev
```

### Run for Production
```sh
npm run start
```
