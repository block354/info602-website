# Client
This is the client for the web application.

## Setup

### Requirements
- [Node.js](https://nodejs.org/en) version 18.3 or higher.

### Installation
### Install Dependencies
```sh
npm install
```

### Compile and Hot-Reload for Development
```sh
npm run dev
```

### Compile and Minify for Production
```sh
npm run build
```
